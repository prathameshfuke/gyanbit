INTRODUCTION

A console you build with code

Most game consoles are made for you to play. GyanBit is made by you — every game, every animation, every sound is written by you in the browser and runs on real electronics.

💻

Browser IDE

Write JavaScript with syntax highlighting, auto-formatting, and a live OLED simulator — no installs needed.

📺

128×64 OLED Simulator

See your code run on a pixel-perfect virtual screen before touching real hardware.

⚡

Flash to Real Hardware

One click generates MicroPython and flashes it via WebSerial to your device. No drivers, no cables frustration.

🎮

7 Built-in Games

Start with Flappy, Space Invaders, Snake, Breakout, Pong, Maze, and Dodger. Study the code, modify them, make them your own.

🔊

8-bit Audio

Square-wave beeps via Web Audio API in the browser, and real PWM audio through a PAM8403 amp on hardware.

📐

bit.* API

Simple, purposeful commands like bit.clear(), bit.fill(), bit.text() designed explicitly for beginners.
