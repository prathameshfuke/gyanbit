What is GyanBit?

A console you build with code

Most game consoles are made for you to play. GyanBit is made by you — every game, every animation, every sound is written by you in the browser and runs on real electronics.

💻

Browser IDE

Write JavaScript with syntax highlighting, bit.* API autocomplete hints, and a live OLED simulator — no installs needed.

📺

128×64 OLED Simulator

See your code run on a pixel-perfect virtual screen with bloom effects and CRT scanlines before touching real hardware.

⚡

Flash to Real Hardware

One click generates MicroPython and flashes it via WebSerial to your Raspberry Pi Pico. No drivers, no cables frustration.

🎮

5 Built-in Games

Start with Snake, Breakout, Pong, Maze, and Dodger. Study the code, modify them, make them your own.

🔊

8-bit Audio

Square-wave beeps via Web Audio API in the browser, and real PWM audio through a PAM8403 amp on hardware.

📐

bit.* API

Simple, purposeful commands: bit.clear(), bit.fill(), bit.text(), bit.beep() — designed for beginners.

How it works

From idea to game in minutes

01

Write in the Studio

Open GyanBit Studio in Chrome. Pick a game from the gallery or write your own using the bit.* API.

→
02

Simulate on OLED

Press RUN. Your code runs in a 30fps loop and renders on the pixel-perfect virtual OLED display instantly.

→
03

Flash & Play

Connect via USB, press FLASH. Your game is converted to MicroPython and runs on the real console forever.
